Drawing Shapes

After effects gives you very basic drawing tools for creating ellipses, polygon, etc.

Changing Attributes

Attributes for a given layer can be found in a set of dropdown menus under the layer name in the timeline panel.

Importing Art from illustrator

Illustrator was created to make complex vector drawings, so it’s the better place to start if you want to create a vector drawing more elaborate than basic shapes. Remember that you can convert illustrator layers to after effects shapes, but the conversion isn’t always perfect. Custom brushes will import as shapes rather than lines, for example. Importing from photoshop is quite similar to illustrator, with each layer appearing in a composition. Best if you are working in pixels (imagine you are animating a photo collage, for example) rather than vector shapes.

The Timeline

The panel on the left is an outline of all your layers, with their attributes in the The panel to the right shows the times where keyframes (a location on a timeline which marks the beginning or end of a transition) are placed.

Animating Attributes

Attributes can be animated with keyframes by clicking the stopwatch icon and changing the values at different points in time.

Attributes can also be animated with expressions. You can add an expression to an attribute by option-clicking the stopwatch icon and typing it into the resulting text box. wiggle([frequency],[amplitude]); randomizes a given attribute loopOut(); loops the keyframes in a given attribute
