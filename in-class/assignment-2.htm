Try to make 50 variations on a pattern

Create a comp with the total number of frames equal to the number of still images you'll need. This one is 50 seconds long, with a frame rate of 1fps.

Create a layout, and then add motion. I used the wiggle() expression to inject randomness into the movements of different elements.

when you're finished, go to render > add to render queue but instead of exporting as a quicktime movie, export as a jpeg sequence.

This export will create a folder containing your 50 images that you can place into your design.

You could use this technique to create a whole library of patterns for use on print collateral. Or create a "living logo" with many different permutations of a single mark.
